# Moon Tasker

月の満ち欠けと生命体育成をテーマにした、没入型タスク管理アプリケーション

## 特徴

- 🌙 **月のサイクル管理**: 新月から満月までPDCAサイクルで目標管理
- 🐾 **生命体育成**: タスク達成で育つクリーチャーシステム
- ⭐ **星座コレクション**: 達成度に応じて解放されるバッジシステム
- ⏱️ **連続タイマー**: タスクと休憩を自動切り替え
- 🤖 **AIスケジュール**: 最適なタスク順序を自動生成
- 🎵 **プレイリスト**: お気に入りのタスクセットを保存・再生

## インストール

```bash
pip install -r requirements.txt
```

## 実行方法

```bash
python -m moon_tasker.main
```

## 技術スタック

- **GUI**: Flet
- **Database**: SQLite
- **言語**: Python 3.x

## プロジェクト構造

```
moon_tasker/
├── main.py              # アプリケーションエントリーポイント
├── database.py          # データベース操作
├── models.py            # データモデル
├── views/               # UI画面コンポーネント
└── logic/               # ビジネスロジック
```
